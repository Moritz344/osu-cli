# Features
- get information about osu! users
- get information about osu! beatmaps
- join chat channels
- get latest changelog

# Run Locally

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project uses the osu! api: https://osu.ppy.sh/docs/index.html#get-user
